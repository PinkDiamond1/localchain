# How to run the examples

## 1. Local Event Listener

### Step 1. Start local event handler

```
node local
```

### Step 2. Run sender

The sender app will mass post transactions to the pool.

```
node sender
```

---

## 2. Global Event Listener

### Step 1. Start local event handler

```
node global
```

### Step 2. Run sender

The sender app will mass post transactions to the pool.

```
node sender
```
